Setup

FAQ

Frequently asked questions

How do I check my Claude AI rate limits?

Install Claude God — it shows your Claude AI rate limits in real time with color-coded gauges in your macOS menu bar. It tracks session (5h), weekly, and per-model (Sonnet, Opus) quotas with live countdown timers showing exactly when limits reset.

How much does Claude Code cost per day?

Claude God tracks your Claude Code spending in real time. It shows daily, weekly, and monthly costs broken down by model (Opus, Sonnet, Haiku) with 30-day trend charts. You can also export your cost data as CSV.

How do I monitor Claude API usage on macOS?

Install Claude God with Homebrew: brew tap lcharvol/tap && brew install --cask claude-god. It reads your existing Claude login credentials — no separate API key needed. It monitors quotas, costs, tokens, and session history.

Does Claude God work with Claude Pro, Max, and Team plans?

Yes. Claude God works with all Claude subscription plans that support Claude Code — including Pro, Max, and Team. It uses the same OAuth credentials as the Claude CLI, so any plan that works with claude login is supported.

How do I know when my Claude quota resets?

Claude God shows live countdown timers for each quota directly in your menu bar. You can also configure threshold-based notifications (e.g. alert at 80% usage) so you know before you hit a rate limit.

Is Claude God safe? Does it send my data anywhere?

Claude God is fully open source (MIT license) with zero telemetry. Your credentials never leave your Mac. It only communicates with the official Claude API to fetch your quota data — the same endpoint used by claude.ai. You can audit every line of code on GitHub.

What is the Claude Code session timeline?

The session timeline lets you browse your Claude Code usage day by day. Each session shows the project name, time range, total cost, and model used. Click to expand and see individual messages with their token counts, model, and cost.
